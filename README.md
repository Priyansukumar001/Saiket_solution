# 🤖 Machine Learning Internship – SaiKet Systems

This repository contains the tasks completed during my **Machine Learning Internship at SaiKet Systems**. The internship focused on data preprocessing, machine learning fundamentals, and predictive modeling using Python.

## 🚀 Tasks Completed

### 🔹 Task 1: Data Preparation

* Loaded and analyzed the Telco Customer Churn Dataset.
* Handled missing values and converted data types.
* Encoded categorical variables.
* Prepared the dataset for machine learning models.

### 🔹 Task 2: Train-Test Split

* Separated features and target variable (`Churn`).
* Split the dataset into **80% training** and **20% testing** sets.
* Used stratified sampling to maintain class distribution.

### 🔹 Task 3: Customer Churn Prediction

* Built a **Logistic Regression** model.
* Predicted customer churn on test data.
* Evaluated the model using Accuracy Score, Confusion Matrix, and Classification Report.
